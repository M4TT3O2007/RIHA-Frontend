# RIHA-Frontend
RIHA frontend (HTML/CSS) on which all else modules are based.

# Getting started
Install dependencies
```sh
$ npm install
```
To watch style changes
```sh
$ npm run watch
```
To build styles
```sh
$ npm run build
```
